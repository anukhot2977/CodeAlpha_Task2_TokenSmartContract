# 💰 CodeAlpha Internship - Task 2

## 📌 Project Title: Token Smart Contract

This project is a Token Smart Contract developed using Solidity as part of the Blockchain Internship at CodeAlpha.

The smart contract simulates a basic ERC-20 style token system where digital tokens can be created, stored, transferred, and managed securely on the blockchain. The total token supply is initially assigned to the contract deployer, and users can transfer tokens between accounts while maintaining transparent balance records.

This project demonstrates how blockchain technology enables secure and decentralized digital asset management and forms the foundation of cryptocurrency systems.

---

## 🚀 Features

✔ Token creation with fixed total supply
✔ Automatic assignment of tokens to contract deployer
✔ Secure transfer of tokens between users
✔ Balance tracking using mappings
✔ Validation using `require()` conditions
✔ Transparent and decentralized asset management

---

## 🛠️ Technologies Used

* Solidity
* Remix IDE
* Ethereum Virtual Machine (Remix VM)
* GitHub

---

## 📂 Smart Contract Functions

### 1. constructor()

Initializes the total token supply and assigns all tokens to the contract deployer.

### 2. transfer()

Allows users to transfer tokens from one account to another securely.

### 3. getBalance()

Returns the token balance of a specific user account.
